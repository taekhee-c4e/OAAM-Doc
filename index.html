Possible data attributes

Data Attribute Default Required Example
Image src data-imgsrc O data-imgsrc="https://oaam.s3..."
Image Width data-width If empty, default width is 250px X data-width="180"
Image Height data-height If empty, default height is set as "auto" X data-height="200"
Image Title data-imgtitle O data-imgtitle="image title"
Image Alt text data-imgalt If image alt text is same as image title, you can skip this X data-imgalt="image alt"

Spotify

You can set whitespace after Spotify Icon
This case there is 5 space after the Spotify Icon 

data-space="5"

You can leave it empty if you do not need space after the Spotify Icon 

data-space=""

Data Attribute Default Required Example
Spotify Title data-spottitle O data-spottitle="SpotTitle"
Whitespace after music icon data-space X data-space="10"
Image Height data-spotid X If there are multiple pieces have same title, get "id" from DF
data-spotid="162"
Custom style data-spotstyle X data-spotstyle="color;red; font-size:20;"
